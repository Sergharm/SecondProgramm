## Описание проекта

Программное средство «Encryption Algorithm RGR» предназначено для:
- Шифрования и дешифрования текстовых данных
- Шифрования и дешифрования файлов (текстовых и бинарных)
- Генерации криптографических ключей
- Демонстрации работы различных алгоритмов шифрования

**Цель работы** — практическое изучение принципов модульного программирования, работы с различными типами шифров и основ криптографической защиты информации.

## Реализованные алгоритмы

### Garmaev_Ciphers

#### Аффинный шифр (Affine Cipher)
Классический шифр подстановки на основе математической функции.

**Формулы:**
- Шифрование: `E(x) = (a*x + b) mod m`
- Расшифрование: `D(x) = a⁻¹*(x - b) mod m`

**Особенности:**
- Работает с символами алфавита
- Ключ состоит из двух чисел (a, b)
- Требует проверки взаимной простоты a и m

#### Шифр Плейфера (Playfair Cipher)
Биграммный шифр подстановки с использованием матрицы.

**Принцип работы:**
- Использует матрицу 5×5 или 16×16
- Шифрует пары букв (биграммы)
- Применяет правила замены в зависимости от расположения

---

### Saids_Ciphers

#### RSA (Rivest–Shamir–Adleman)
Асимметричный криптографический алгоритм.

**Принцип:**
- Генерация пары ключей (открытый и закрытый)
- Основан на сложности факторизации больших чисел
- Использует модульную арифметику

**Применение:**
- Шифрование файлов
- Цифровые подписи

#### A5/1
Поточный шифр, используемый в GSM.

**Особенности:**
- Три регистра сдвига с обратной связью
- Генерация псевдослучайной гаммы
- Шифрование голосового трафика

#### AES-128 CFB
Современный блочный шифр в режиме CFB.

**Параметры:**
- Длина ключа: 128 бит
- Режим: Cipher Feedback
- Преобразует блочный шифр в поточный

### Nesterov_Siphers

#### Шифр Хилла (Hill Cipher)
Матричный блочный шифр.

**Формула:** `C = K * P mod 256`

**Требования:**
- Матрица должна быть обратимой по модулю 256
- Определитель матрицы должен быть нечётным

#### Шифр Вернама (Vernam Cipher)
Поточный шифр с абсолютной стойкостью.

**Принцип:**
- Использует гамму той же длины, что и сообщение
- Шифрование: `C = (P + K) mod m`
- Расшифрование: `P = (C - K) mod m`
- Ключ используется один раз (одноразовый блокнот)

---

### Rozhkov_Ciphers

#### Шифр ADFGX
Классический шифр Первой мировой войны.

**Принцип:**
1. Фракционирование: каждый байт → 2 Hex-цифры → 4 буквы ADFGX
2. Столбцовая перестановка по ключевому слову

**Особенности:**
- Поддержка UTF-8 (русский язык, эмодзи)
- Hex-интерфейс для удобного копирования

#### Шифр Бифида (Bifid Cipher)
Комбинированный шифр (подстановка + перестановка).

**Принцип:**
- Создание координатной сетки для Hex-символов
- Выписывание координат строк и столбцов
- Перемешивание координат
- Обратная сборка по новым координатам
